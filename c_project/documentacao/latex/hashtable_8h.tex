Insere ou atualiza um artigo na hashtable de utilizadores. 

Copiar strings porque o parser pode apagá-\/las e a hash table só guarda um pointer. Copiar longs usadas como key também, porque keys são usadas a partir dos seus pointers. Criar nova revisão (necessária caso o artigo já exista ou não). Informar se o artigo foi encontrado ou não através de article\+Found e article\+Updated. Criar ou atualizar o artigo dependendo se este foi encontrado ou não.
